🧮 PLACEMENT READINESS SCORE (PRS)

Define formula strictly:

PRS =
(0.5 × coding_score) +
(0.3 × mock_score) +
(0.2 × resume_score)

Create function:

recalculatePRS(studentId)

This must:

Fetch coding_score

Fetch mock_score

Fetch resume_score

Update pr_score field

Call this function whenever:

Interview attempt completed

Resume scored

Mentor gives performance rating

🏢 COMPANY-WISE INTERVIEW SYSTEM

FLOW:

Student selects company from dropdown.

System fetches all questions from that company.

Display one question per page.

Provide:

Question text

Topic tag

Answer textarea

Submit button

On submission:

Compare student answer with correct_answer (case-insensitive)

If match → full marks

Else → 0 marks (MVP level)

Save result in ATTEMPT_DETAILS

After last question:

Calculate total_score

Save in STUDENT_ATTEMPTS

Update student coding_score:
coding_score = average of last 5 attempts

Call recalculatePRS()

Display final result page:

Total score

Correct answers count

Weak topics

Improvement suggestion

UI must be clean and minimal.
